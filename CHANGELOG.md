# Change Log

All notable changes to the "Objectify Params" extension will be documented in this file.

## [0.1.0] - 2025-11-24

### Initial Release
- Convert function parameters to object destructuring syntax
- Automatic workspace-wide refactoring of function signatures and call sites
- Smart detection of safe vs uncertain conversions
- Interactive preview dialogs for uncertain cases
- Support for TypeScript, JavaScript, Vue, and Svelte files
- Configurable file include/exclude patterns
- Optional monitoring mode to review all conversions
- Adjustable highlight delay for previews
- Type-safe conversions preserving TypeScript types and optional parameters
